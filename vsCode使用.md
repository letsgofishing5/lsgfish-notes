### 个人使用符号

&&：表示连在一起使用

### 常用快捷键

#### 1、打开终端 

```bash
ctrl + `
```

#### 2、打开设置

```bash
ctrl + ,
```

#### 3、自定义快捷键

```bash
ctrl + k && ctrl + s
```

#### 4、格式化文档

```bash
alt + shift + F
```

#### 5、运行命令

```bash
ctrl + shift + p
```

#### 6、跳转到指定行

```bash
ctrl + G
```

#### 7、替换

```bash
ctrl + h
```
