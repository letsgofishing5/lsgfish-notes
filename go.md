## 配置环境变量

1. GOPATH：保存的是以后项目的存放目录GO
2. GOROOT：保存的是安装包的目录
3. Path：go的bin目录
4. 在步骤一的GOPATH目录中，新建：src、pkg、bin

#### 常用命令

1. go env
2. go run
3. go build
4. go build -o hello.exe  //给编译后的文件重命名
